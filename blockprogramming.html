Block Programming: Weekend Junior Registration (Ages 10 and below)

Give your child the opportunity to explore game development with our MakeCode Arcade course, tailored for kids under 10. Starting with the basics, your child will learn how to control game elements and design interactive environments. As they advance, they’ll dive into more complex concepts like sprite interactions, game physics, and level design. By the end of the course, your child will have created a fully functional game, applying everything they’ve learned into a final project. This course not only nurtures creativity but also builds critical thinking and problem-solving skills, laying a strong foundation for future tech-savvy endeavors.

Course Breakdown:



Level 1


→ Introduction to MakeCode Arcade: Understanding the game loop and basic sprite manipulation.

→ Working with Tiles and Tilemaps: Introduction to tilemaps and environment design.

→ Basic Events and Interactions: Handling events and sprite interactions.

→ Variables and Conditionals: Using variables and conditional statements to control game logic.


Level 2


→ Advanced Sprite Interactions: Managing sprite groups, more complex interactions, and animations.

→ Arrays and Loops: Using arrays to manage multiple objects and loops to create repetitive actions.

→ Game Physics Basics: Applying basic physics principles like gravity and velocity to game objects.

→ Timers and Delays: Utilizing timers for controlling event timing and game pacing.


Level 3


→ Advanced Tilemaps and Level Design: Designing complex levels and implementing level transitions.

→ Creating a Heads-Up Display (HUD): Implementing and managing HUD elements and overlays.

→ Custom Assets and Animations: Creating and integrating custom assets and more sophisticated animations.

→ Game Polishing and Testing: Incorporating sound, debugging, and refining gameplay mechanics.


Level 4


→ Scene Management and Game Structure: Managing multiple game scenes and states, including menus and cutscenes.

→ Advanced Game Logic and AI: Developing complex AI behaviors and optimizing game performance.

→ Final Project Development: Integrating all learned concepts into a cohesive final game project.

→ Game Presentation and Showcase: Preparing and presenting a polished game, demonstrating understanding of all key concepts.
