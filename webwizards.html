Course Breakdown:



Level 1: Introduction to Web Design


What Kids Will Learn:

→ Basics of how websites work

→ Understanding HTML, CSS, and JavaScript.

Activities:

→ Create a simple webpage with text and images.

→ Begin personalizing their page with colors and fonts.


Level 2: HTML Mastery


What Kids Will Learn:

→ Structure of an HTML document.

→ Using tags to create headings, paragraphs, and links.

Activities:

→ Design a webpage about their favorite hobby or interest.

→ Add links to other favorite websites.


Level 3: Getting Stylish with CSS


What Kids Will Learn:

→ Changing the look of a website with CSS.

→ Introduction to colors, fonts, and layouts.

Activities:

→ Style their webpage with vibrant colors and cool fonts.

→ Learn to arrange elements using basic layout techniques.


Level 4: Advanced CSS Techniques


What Kids Will Learn:

→ Understanding the box model, margins, padding, and borders.

→ Introduction to flexbox and grid layouts

Activities:

→ Create a visually appealing blog layout.

→ Experiment with advanced layouts to make their site unique


Level 5: Introduction to JavaScript


What Kids Will Learn:

→ Basics of JavaScript and how it makes websites interactive.

→ Simple functions and events in JavaScript.

Activities:

→ Add interactive buttons to their website.

→ Create simple games like 'Click the Button' on their site.


Level 6: Making Interactive Web Pages


What Kids Will Learn:

→ Deep dive into JavaScript functions and events.

→ Creating interactive elements like pop-ups and animations.

Activities:

→ Build a 'Guess the Number' game on their website.

→ Add animations that respond to user actions.


Level 7: Building a Mini Project


What Kids Will Learn:

→ Combining HTML, CSS, and JavaScript to create a complete website.

→ Planning and designing a web project from start to finish.

Activities:

→ Start building their personal mini-website.

→ Incorporate everything learned to make the site interactive and stylish.

Level 8: Project Showcase


What Kids Will Learn:

→ Finalizing their projects with last-minute tweaks and designs.

→ Presenting their work to classmates and parents.

Activities:

→ Final project presentations.

→ Celebrate your child’s achievements with a graduation ceremony and certificate!
